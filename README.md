# Installation
Install the stable version from Bioconductor
```
## try http:// if https:// URLs are not supported
source("https://bioconductor.org/biocLite.R")
biocLite("MODA")
```

Install the developer version from github
```
library(devtools)
install_github("fairmiracle/MODA")
```
